## Docs source

Docs live at `packages/docs` in `different-ai/maya`.

## Run locally

```bash
npm i -g mint
cd packages/docs
mint dev
```

## Validate docs

```bash
mint broken-links
```

## Typical workflow

1. Edit `.mdx` pages or `docs.json` in `packages/docs`.
2. Validate with local preview and `mint broken-links`.
3. Push to `different-ai/maya`.
4. Confirm Mintlify Git settings point to:
   - repo: `different-ai/maya`
   - monorepo path: `/packages/docs`
